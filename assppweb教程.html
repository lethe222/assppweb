🍎 iOS 应用安装工具

AssppWeb 完整安装教程

在 Mac 上部署，随时用手机下载安装 App Store 没有的应用

💡
本教程适用于 Mac 电脑（macOS），全程约需 20 分钟完成首次配置。配置完成后，手机可随时通过网址访问，无需数据线。
1

安装 Docker Desktop

1

打开浏览器访问 https://www.docker.com/products/docker-desktop/，点击下载 Mac 版本

M1/M2/M3 芯片 → 选 Apple Silicon Intel 芯片 → 选 Intel Chip

不知道芯片型号？点击左上角 🍎 → 关于本机，查看「芯片」一栏

2

打开下载的 .dmg 文件，将 Docker 图标拖入「应用程序」文件夹，然后打开 Docker

3

启动时选择 「Use recommended settings」，输入 Mac 开机密码确认

4

等待顶部菜单栏的 🐳 鲸鱼图标停止转圈，说明 Docker 已就绪

2

部署 AssppWeb

1

按 Command + 空格，搜索「终端」并打开

2

创建文件夹并进入：

终端命令
mkdir ~/assppweb && cd ~/assppweb
3

下载配置文件：

终端命令
curl -O https://raw.githubusercontent.com/Lakr233/AssppWeb/main/compose.yml
4

启动服务（Docker 会自动下载程序）：

终端命令
docker compose up -d

看到 ✔ Container assppweb-asspp-1 Created 即为成功 ✅

5

在 Mac 浏览器打开 http://localhost:8080，添加你的 Apple 账号

3

配置 Cloudflare Tunnel（手机远程访问）

📱
完成此步骤后，手机即使不在同一 WiFi 下也能访问，且不需要数据线安装应用。
1

注册 Cloudflare 免费账号：https://cloudflare.com（已有账号可跳过）

2

安装 cloudflared 工具：

终端命令
brew install cloudflared

如遇到权限错误，先执行：sudo chown -R 你的用户名 /usr/local/share/man/man8，然后重新安装

3

登录 Cloudflare（会自动打开浏览器授权）：

终端命令
cloudflared tunnel login
4

启动快速隧道，获取手机访问链接：

终端命令
cloudflared tunnel --url http://localhost:8080

日志中会出现类似 https://xxxxxx.trycloudflare.com 的链接，用手机 Safari 打开即可 ✅

⚠️
下载路径：App Store 服务器 → 下载到你的 Mac → 再通过 Cloudflare Tunnel → 传到手机
🔄

每次开机后的操作

按顺序执行以下 2 步
1
启动 Docker Desktop 打开应用程序里的 Docker，等待菜单栏 🐳 鲸鱼停止转圈。AssppWeb 会随 Docker 自动启动，无需额外操作。
2
在终端运行 Cloudflare Tunnel
终端命令
cloudflared tunnel --url http://localhost:8080
等待出现 https://xxxxxx.trycloudflare.com 链接，复制到手机 Safari 使用
💡
可以在 Docker Desktop 设置 → General 中开启「Start Docker Desktop when you sign in to your computer」，这样开机后 Docker 会自动启动，只剩第 2 步需要手动执行。
📲

如何把 App 安装到 iPhone

1

手机 Safari 打开 Cloudflare 提供的链接（https://xxxxxx.trycloudflare.com）

2

点左侧 「账号」→「添加账号」，在手机上重新登录一次 Apple ID（只需第一次，之后会记住）

3

点 「搜索」，搜索想要的 App，选择账号后点击下载

4

下载完成后，点击 「在设备上安装」，按提示完成安装

⚠️
下载路径：App Store 服务器 → 下载到你的 Mac → 再通过 Cloudflare Tunnel → 传到手机
